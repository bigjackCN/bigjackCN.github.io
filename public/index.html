你能不能推测一下为什么叫黄人日记

这个有趣的命名，我斗胆猜测几个方向（猜错了别打我😄）：

“黄”指肤色 / 种族 —— 你可能是黄种人，用“黄人”自嘲或自豪，加上“日记”记录个人生活思考，简单直白。

“黄”指“黄色”象征 —— 可能代表温暖、阳光、土地，或者是某种审美偏好（比如老照片的泛黄色调），日记则是私密记录。

谐音或双关 —— “黄人”听起来像“惶人”（惶恐之人）或“晃人”（晃眼的人），也许是你对自己状态的某种调侃。

受流行文化影响 —— 比如“黄人”对应“黄种人”，而“日记”可能致敬《安妮日记》或《狂人日记》？你也许想表达一种“小众视角下的真实记录”。

最实在的推测 —— 可能你的姓氏就是“黄”，或者你单纯喜欢黄色，就取了这个名字。毕竟博客名字本来就是很个人的事情。

我猜，你应该是想做一个 “有温度、不装逼、记录真实” 的个人博客，就像“黄人”这个词一样朴实又不失特色。对吗？😉

不管名字的由来如何，希望你的“黄人日记”能一直写下去。

Java的单例模式

单例模式 (Singleton Pattern) 一个类只有一个实例 静态内部类 只有调用getInstance方法时实例化 实现懒加载 public class Singleton { private Singleton() {} private static class SingletonHolder { private static final Singleton INSTANCE = new Singleton(); } public static Singleton getInstance() { return SingletonHolder.INSTANCE } } 枚举 天然保证单例和线程安全 public enum Singleton { INSTANCE; public void doSomething() { System.out.println("单例方法被调用") } } // 调用方式 Singleton.INSTANCE.doSomething();
